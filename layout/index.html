Set Interval Reading and Turn ON/OFF Data Logging (in Min)






INFO

Some Information:

To set an Interval between the Reading of the Sensors connected into ESP32, you can just select on the left side and set On the Data Logging.

You can see the state of Data Logging on the right side, if it's green "Data logging enabled", it means that ESP32 is sending data by its sensors to the Database, if it's red "Data logging disabled", it means it's turned on but ESP32 won't send any data do the Database.
